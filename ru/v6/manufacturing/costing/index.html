Себестоимость: как рассчитывается

Этот раздел описывает, как в системе рассчитывается себестоимость изделия при выполнении производственного заказа.

Описание составлено по логике, реализованной в модуле расчёта себестоимости производственных заказов.

Общая идея​

Себестоимость считается по факту:

  1. Сначала фиксируется фактическое потребление материалов по производственному заказу.
  2. По факту потребления формируется стоимость списания материалов.
  3. Сумма списаний (плюс дополнительные и трудовые затраты, если они заданы) образует себестоимость производственного заказа.
  4. Себестоимость заказа распределяется на выпуск (на строки выпуска) по коэффициентам распределения.

Что участвует в расчёте​

1) Фактическое потребление материалов​

В производственном заказе есть строки материалов. Для каждой строки фиксируется фактическое количество «Потреблено».

В расчёт себестоимости попадают только строки, где:

  • фактическое потребление заполнено;
  • номенклатура является материальной (участвует в учёте запасов).

2) Место хранения материалов и дата выполнения​

Стоимость потребления определяется с учётом:

  • места хранения материалов (место хранения, из которого идёт списание);
  • даты выполнения производственного заказа.

Практический смысл:

  • если в разных местах хранения разная себестоимость, списание будет оценено по месту хранения материалов;
  • дата выполнения фиксирует «момент оценки», то есть по какой дате берётся стоимость.

3) Дополнительные и трудовые затраты​

К себестоимости производственного заказа могут добавляться:

  • Дополнительные затраты;
  • Трудовые затраты.

Эти суммы участвуют только в том случае, если они настроены и заполнены в вашей конфигурации.

Формулы​

Себестоимость материалов​

Себестоимость материалов по заказу — это сумма стоимостей списаний по всем строкам материалов.

Иными словами:

  • для каждой строки материалов считается сумма списания;
  • затем суммы складываются в себестоимость заказа.

Итоговая себестоимость производственного заказа​

Итоговая себестоимость заказа складывается из:

  1. себестоимости материалов;
  2. дополнительных затрат;
  3. трудовых затрат.

Как себестоимость распределяется на выпуск​

В заказе есть строки выпуска (какие изделия/полуфабрикаты произведены). Себестоимость заказа распределяется между строками выпуска.

Коэффициент распределения​

Для строки выпуска может быть заполнен коэффициент «Коэффициент распределения себестоимости».

Расчёт доли строки:

  • если коэффициенты заполнены хотя бы у одной строки, то доля строки = коэффициент строки / сумма коэффициентов по заказу;
  • если коэффициенты не заполнены, то вся себестоимость относится на основное изделие заказа (на строку выпуска, где номенклатура совпадает с номенклатурой заказа).

Стоимость выпуска​

Стоимость, которая будет отнесена на конкретную строку выпуска:

стоимость строки выпуска = итоговая себестоимость заказа × доля строки

Себестоимость единицы​

Чтобы получить себестоимость единицы по строке выпуска:

себестоимость единицы = стоимость строки выпуска / фактически произведено по строке

Если фактически произведено равно нулю, себестоимость единицы рассчитать нельзя — сначала зафиксируйте выпуск.

Где посмотреть себестоимость​

Обычно себестоимость видна:

  • в списке производственных заказов — поля «Себестоимость», «Дополнительные затраты», «Трудовые затраты», «Итоговая себестоимость»;
  • в отчетах по производству (например, в отчете по заказам, если он включён в разделе отчетности).

Типовые ситуации и проверки​

Итоговая себестоимость равна нулю​

Проверьте:

  • заполнено ли «Потреблено» по строкам материалов;
  • выбрано ли место хранения материалов;
  • задана ли дата выполнения (обычно появляется при завершении заказа).

Себестоимость «вся ушла» на одну строку выпуска​

Это нормально, если коэффициенты распределения не заполнены. Тогда система относит всю сумму на основное изделие.

Себестоимость распределилась «не так, как ожидается»​

Проверьте:

  • коэффициенты распределения по строкам выпуска;
  • фактически произведенные количества;
  • не добавлены ли дополнительные/трудовые затраты.
