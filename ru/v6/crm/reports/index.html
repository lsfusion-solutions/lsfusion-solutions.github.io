Отчётность

Отчет по лидам​

Где находится​

Откройте раздел «Отчетность», затем «Отчет по лидам».

Назначение​

Отчет предназначен для анализа лидов в разрезах времени и атрибутов. Он помогает ответить на вопросы:

  • сколько лидов поступило за период;
  • какие статусы встречаются чаще;
  • какая ожидаемая сумма и как она распределена по менеджерам;
  • какие лиды планировались к закрытию в определённый период.

Отчет построен в виде сводной таблицы: вы можете менять разрезы и группировки в зависимости от задачи.

Какие поля доступны​

В отчете обычно есть:

  • основные атрибуты: Идентификатор, Наименование, Статус лида, Покупатель, Менеджер по продажам, Приоритет лида, Ярлыки лида;
  • маркетинг (если включён модуль маркетинга): Кампания, Канал, Источник;
  • прогноз: Вероятность, Ожидаемое закрытие, Ожидаемая сумма;
  • контакты: Телефон, Электронная почта;
  • описание и дополнительные данные (адрес, сайт, наименование организации, контактное лицо) — если они заполнены.

Фильтр «Интервал дат»​

В блоке «Фильтры» доступен параметр «Интервал дат».

Он ограничивает выборку лидов по дате лида:

  • не раньше выбранной даты «с»;
  • не позже выбранной даты «по».

Рекомендация: при анализе воронки за месяц всегда задавайте интервал дат, чтобы отчет был быстрым и сравнимым.

Временные разрезы​

В отчете доступны вычисляемые временные разрезы:

  • по дате лида: минута, час, дата, день недели, неделя, месяц, год;
  • по ожидаемому закрытию: день недели, неделя, месяц, год.

Это позволяет, например, сравнить распределение ожидаемых закрытий по неделям или оценить загрузку менеджеров по месяцам.

Примеры полезных разрезов​

Ниже — примеры вопросов, которые удобно решать этим отчетом:

  1. Сколько лидов поступает по неделям
    • установите интервал дат;
    • сгруппируйте по неделе (по дате лида);
    • посчитайте количество лидов в каждой группе.
  2. Планируемые закрытия по месяцам
    • установите интервал дат;
    • сгруппируйте по месяцу ожидаемого закрытия;
    • анализируйте ожидаемую сумму.
  3. Нагрузка по менеджерам
    • сгруппируйте по менеджеру по продажам;
    • добавьте разрез по статусу лида;
    • сравните ожидаемую сумму и количество.
  4. Эффективность маркетинга
    • сгруппируйте по кампании, каналу или источнику;
    • анализируйте количество лидов и ожидаемую сумму.

Рекомендации по качеству данных​

Чтобы отчет был полезным:

  • заполняйте «Ожидаемую сумму» и «Ожидаемое закрытие» хотя бы для лидов в работе;
  • следите, чтобы у лидов был назначен менеджер по продажам;
  • не оставляйте лиды в одном статусе без движения: переводите по этапам или закрывайте.
